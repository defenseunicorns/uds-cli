The [UDS CLI](https://github.com/defenseunicorns/uds-cli) is the primary interface for deploying and managing UDS bundles and mission applications. It simplifies deployment by bundling multiple Zarf packages into a single deployable artifact and extends that capability at runtime through Pepr.
